# Proposal: AI Chat with Presets

## Summary
Экран чата с AI-ассистентом в TabBar. Пресеты для быстрых запросов. История чата сохраняется в БД — юзер всегда может перечитать ответы.

## Motivation
Подготовка инфраструктуры для интеграции с DeepSeek API. UI-скелет + API endpoints готовы для подключения AI в будущем.

## Scope
- Backend: ChatModule (ChatMessage model, endpoints)
- Mobile: /main/chat screen в TabBar, пресеты-кнопки

## Non-goals
- Интеграция с DeepSeek (future)
- Голосовой ввод
- Сканирование чеков

## Пресеты
1. **Отчёт по тратам за период** — анализ расходов за день/неделю/месяц
2. **Анализ бюджета** — выполнение бюджетов, рекомендации
3. **Как сохранить больше** — практические советы по экономии
4. **Отслеживать динамику** — изменения в финансовом поведении

## Хранение сообщений
- `ChatMessage.role = USER` — сообщение юзера (сохраняется сразу)
- `ChatMessage.role = ASSISTANT` — ответ DeepSeek (сохраняется когда пришёл)
- `presetType` — тип пресета если был использован
- Юзер может очистить всю историю (DELETE /chat/messages)
- История доступна между сессиями
