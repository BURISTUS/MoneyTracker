. Концепция: «Побег из общества потребления»

Лор (Сюжет): Пользователь начинает игру в статусе «Потребитель» (Consumer Drone). Его аватар обвешан пакетами, логотипами брендов, он выглядит уставшим. Вокруг него — серый, замусоренный город. Цель: Достичь статуса «Архитектор» (Financial Architect). Аватар становится минималистичным, свободным, вокруг него — чистое футуристичное пространство.
2. Механика «Кладбище импульсов» (Anti-Spend Impulse)

Это главная киллер-фича для борьбы с «трендовыми шмотками».

Как это работает: Когда пользователь хочет купить что-то ненужное (но хочется), он не покупает, а заносит это в «Инкубатор желаний» (Wishlist with Timer).

    Таймер «Остывания»: Приложение блокирует кнопку «Купил» для этой вещи на 7 дней (настраиваемо).

    Трансформация: Через 7 дней приложение спрашивает: «Ты все еще хочешь эти кроссовки за 15 000р?»

    Награда за отказ:

        Если юзер нажимает «Мне это не нужно», приложение взрывает этот предмет с красивой анимацией.

        Моментальный дофамин: На счет «Виртуальной свободы» начисляется XP, равное стоимости вещи.

        Визуализация: Приложение показывает: «Ты не купил кроссовки. Эти деньги отправлены в инвестиции. Через 10 лет это будет 45 000 рублей (с учетом сложного процента)».

3. Механика «Цена твоей жизни» (Time-is-Money Converter)

Чтобы отбить желание покупать ерунду, нужно конвертировать цену не в рубли, а в часы жизни.

Настройка: При регистрации юзер вводит зарплату и часы работы. Мы считаем его Real Hourly Rate (реальную стоимость часа).

UX сценарий: Юзер вводит: «Новый iPhone, 120 000 руб». Приложение (вместо "ОК"): «Внимание! Это 160 часов твоей работы. Это 20 рабочих дней. Ты готов просидеть в офисе целый месяц ради этого телефона?» Кнопки: [Все равно купить] (красная, маленькая) и [Я не раб вещей] (зеленая, большая).
4. Ачивки и Прокачка (RPG System)

Не используйте стандартные звездочки. Используйте уровни осознанности.

Уровни:

    Хомяк в колесе (Есть долги, нет накоплений).

    Аскет (Начал закрывать долги, отказался от 5 импульсивных покупок).

    Стратег (Есть подушка безопасности).

    Капиталист (Пассивный доход покрывает 10% трат).

    Свободный человек (Финансовая независимость).

Примеры ачивок (Achievements):

    🏆 «Убийца маркетинга»: Не покупал одежду 3 месяца (кроме категории «База»).

    ☕ «Кофеин-детокс»: Не покупал кофе на вынос 30 дней. Сэкономлено: 6000р.

    📉 «Шорт трендов»: Отказался от покупки вещи, которая была в «Инкубаторе» 2 недели.

5. Челленджи (Социальная механика)

Если это семейный бюджет, добавьте соревновательный элемент.

    Режим «No Spend Month»: Кто из супругов потратит меньше на категорию «Личные хотелки»?

    Визуализация: Два прогресс-бара. Кто победил — тот выбирает фильм на вечер (или приложение само выдает награду-бейджик).

6. Техническая реализация (Для ТЗ)
Backend (NestJS):

Вам понадобятся новые сущности и логика.
Фрагмент кода

// Модель для отложенных желаний
model WishlistItem {
  id        String   @id @default(uuid())
  userId    String
  name      String   // "Dyson Styler"
  price     BigInt
  createdAt DateTime @default(now())
  status    String   // PENDING, REJECTED (Victory!), PURCHASED (Defeat)
  cooldownEnds DateTime // Когда можно принимать решение
}

// Модель игрового профиля
model UserGamification {
  userId          String @unique
  xp              Int    @default(0) // Опыт
  level           Int    @default(1)
  savedAmount     BigInt // Сумма всех REJECTED желаний (виртуальный капитал)
  lifeTimeValue   Int    // Стоимость часа работы юзера (в копейках)
}

Логика (Service):

    GamificationService: При status: REJECTED берет price, добавляет к savedAmount, начисляет XP по формуле (например, 1 XP за каждые 100 рублей). Проверяет LevelUp.

Frontend (React Native):

    Анимации (Lottie):

        Нам нужна библиотека lottie-react-native.

        Найти анимации: разрушение предмета, рост дерева (инвестиций), освобождение от цепей.

    Интерфейс «Инкубатора»:

        Список карточек с таймерами обратного отсчета.

        Пока таймер тикает — карточка «заморожена» (покрыта льдом визуально).

    Конвертер валют:

        При вводе любой суммы рядом мелким шрифтом писать: ≈ 4ч 20мин жизни. Это должно быть сквозной фичей во всем приложении.

Резюме по стратегии

Ваше приложение не говорит «Не трать». Оно говорит: «Не меняй свою жизнь на мусор».

Это очень сильный маркетинговый посыл для Freemium.

    Бесплатно: Учет, базовые уровни.

    Premium: Полный доступ к «Инкубатору желаний» (с аналитикой сэкономленного), расчет стоимости часа жизни, продвинутые ачивки и скины для аватара.

Если реализуете «Цену жизни» и «Кладбище импульсов» — это будет виральная бомба. Люди будут скринить это и постить в сторис: "Прикиньте, приложение сказало мне, что эта сумка стоит неделю моей работы. Передумала брать".
