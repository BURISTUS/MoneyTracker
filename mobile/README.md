# Money Tracker Mobile App

## Описание

Мобильное приложение для финансового трекера "Escape from Consumer Society" - помогает бороться с импульсивными покупками через геймификацию.

## Технологии

- **Expo** + **React Native** - кроссплатформенная разработка
- **Tamagui** - UI компоненты
- **Expo Router** - навигация
- **TanStack Query** - управление состоянием данных
- **Zustand** - локальное состояние
- **Lottie** - анимации
- **Expo Notifications** - push-уведомления

## Установка

```bash
# Перейдите в директорию mobile
cd mobile

# Установите зависимости
npm install

# Запустите приложение
npm start
```

## Экраны

- **Главная** - Дашборд с финансовым прогрессом
- **Транзакции** - Список доходов и расходов
- **Инкубатор** - Wishlist с 7-дневным таймером
- **Прогноз** - Финансовые прогнозы
- **Профиль** - Настройки и статистика

## Особенности

### Инкубатор желаний
Главная фича приложения - 7-дневный таймер перед покупкой:
1. Добавьте желание в инкубатор
2. Подождите 7 дней
3. Решите - нужно ли оно вам на самом деле
4. Получите XP за отказ от покупки

### Цена жизни
Конвертация стоимости покупок в часы вашей работы для осознанного принятия решений.

### Геймификация
- XP за отказ от импульсивных покупок
- Уровни от "Потребителя" до "Архитектора"
- Достижения за различные достижения
