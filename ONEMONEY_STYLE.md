# MoneyTracker - 1money Style UI

## Что сделано

Полностью переработаны экраны создания транзакций и категорий в стиле популярного приложения 1money.

### Создание транзакций (`/main/transactions/create`)

**Особенности:**

1. **Переключатель типа транзакции**
   - Две большие кнопки: "− Расход" и "+ Доход"
   - Цветовая кодировка: красный для расходов, зеленый для доходов
   - Активное состояние с фоном и рамкой

2. **Дисплей суммы**
   - Огромный шрифт (56px) для легкого чтения
   - Форматирование с разделителями разрядов (1 000 ₽)
   - Цвет зависит от типа транзакции

3. **Выбор счета**
   - Горизонтальный скролл
   - Карточки с названием счета и балансом
   - Выделение выбранного счета цветом и рамкой
   - Поддержка всех типов счетов: наличные, карта, кредит, инвестиции

4. **Выбор категории**
   - Сетка из 3 колонок
   - Эмодзи иконки для каждой категории
   - Фильтрация по типу транзакции (только доходы или только расходы)
   - Кнопки с цветовой индикацией при выборе

5. **Поле заметки**
   - Мультилайн текстовое поле
   - Placeholder для подсказки
   - Минималистичный дизайн

6. **Нумпад (цифровая клавиатура)**
   - Кнопки 1-9, точка, 0, удаление
   - Круглые кнопки (60x60px)
   - Высокий контраст для легкого нажатия
   - Кнопка удаления красного цвета

7. **Быстрые суммы**
   - Пресеты: 100, 500, 1000, 5000, 10000, 50000 ₽
   - Горизонтальный скролл
   - Одно нажатие для установки суммы

8. **Фиксированная кнопка сохранения**
   - Всегда внизу экрана
   - Цвет зависит от типа транзакции
   - Отключена, если не все поля заполнены
   - Индикация загрузки

### Создание категорий (`/main/categories/create`)

**Особенности:**

1. **Переключатель типа**
   - "Расход" и "Доход"
   - Красная/зеленая цветовая схема

2. **Поле названия**
   - Большое поле ввода
   - Placeholder "Введите название..."
   - Минималистичный стиль

3. **Выбор иконки**
   - Сетка из 15 иконок для расходов
   - Сетка из 10 иконок для доходов
   - Эмодзи для универсальности
   - Выделение выбранной иконки рамкой и фоном

4. **Выбор цвета**
   - Палитра из 6 цветов для каждого типа
   - Круглые кнопки (60x60px)
   - Галочка на выбранном цвете
   - Специфические цвета для типа:
     - Расходы: красный, оранжевый, желтый, зеленый, голубой, синий
     - Доходы: зеленый, голубой, синий, фиолетовый, розовый, красный

5. **Фиксированная кнопка создания**
   - Всегда внизу экрана
   - Цвет зависит от типа категории
   - Отключена, если не все поля заполнены
   - Индикация загрузки

## Технические улучшения

### Фронтенд

1. **Новый экран транзакций**
   - Полностью переписан с нуля
   - Используется нативный нумпад вместо системной клавиатуры
   - Оптимизированный UX для быстрого ввода

2. **Новый экран категорий**
   - Минималистичный дизайн
   - Большие кнопки для легкого нажатия
   - Интуитивный интерфейс

3. **Исправлен dataStore**
   - `addTransaction` теперь использует переданный `accountId`
   - Правильное форматирование суммы (в копейках для бэкенда)

### Бэкенд

Бэкенд не требует изменений - все API endpoints уже поддерживают:
- Создание транзакций с accountId и categoryId
- Создание категорий с type, icon, color
- Обновление балансов счетов автоматически

## Типизация

Все типы строго определены:
- `Transaction` с полями accountId, categoryId, amount (копейки)
- `Category` с полями type (INCOME/EXPENSE), icon (string), color (hex)
- Полная поддержка TypeScript без `any`

## Тестирование

Для тестирования:

1. **Создание транзакции**
   - Запустите бэкенд: `cd backend && yarn start:dev`
   - Запустите мобильное приложение: `cd mobile && yarn start`
   - Перейдите на экран создания транзакции
   - Выберите тип, введите сумму через нумпад
   - Выберите счет и категорию
   - Добавьте заметку (опционально)
   - Нажмите "Сохранить"

2. **Создание категории**
   - Перейдите на экран создания категории
   - Выберите тип (расход/доход)
   - Введите название
   - Выберите иконку и цвет
   - Нажмите "Создать категорию"
