## 💰 ФИНАНСОВАЯ МОДЕЛЬ: UNIT-ECONOMICS + 3 СЦЕНАРИЯ (SOLO)

### 4.1. Unit Economics (Base Case Assumptions)

```
• CAC (blended): ₽320 (ASO + контент + рефералы)
• Activation Rate (signup→active): 68%
• Freemium→Premium конверсия: 3.7% (месяц 3+)
• Premium ARPU: ₽350/мес ($3.99)
• AI Premium ARPU: ₽690/мес ($7.99)
• Churn (Premium): 4.2%/мес (отраслевой бенчмарк для finance apps)
• LTV = ARPU × Gross Margin × (1 / Monthly Churn)
         = 350 × 0.85 × (1 / 0.042) ≈ ₽7,083
• LTV:CAC = 7,083 / 320 ≈ 22.1 → ✅ Здоровая модель

С AI Premium:
• LTV (AI tier) = 690 × 0.85 × (1 / 0.035) ≈ ₽16,757
• LTV:CAC = 16,757 / 320 ≈ 52.4 → ✅✅ Отличная модель

СОЛО-ПРЕИМУЩЕСТВО:
• Нет расходов на зарплаты (экономия ₽9.5 млн/год)
• Gross Margin: 90–95% (вместо 85% с командой)
• Burn Rate: $200–500/мес (вместо $15–18K/мес)
```
